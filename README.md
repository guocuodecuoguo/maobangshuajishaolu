适用于华为MA5671A、诺基亚G-010S-A、阿尔卡特G-010S-P等SFP猫棒刷机
功能：
板载切换开关支持不同品牌猫棒：H_华为、N_诺基亚、阿尔卡特选N
支持TTL功能：TTL输出有切换开关电压选择3.3V or 5V，3.3V最大输出1A。不刷猫棒也可以当作TTL刷机工具使用，减少闲置和浪费
板载自恢复保险丝，V2.0增加模块电源开关，刷机更方便
